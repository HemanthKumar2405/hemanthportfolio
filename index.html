What I do

Static Websites

Static websites are simple web pages where the content remains fixed and doesn’t change dynamically based on user interaction. They’re built using basic HTML, CSS, and sometimes JavaScript, making them quick to load and ideal for smaller projects like portfolios or informational pages.

Responsive Web design

Responsive websites are designed to adapt seamlessly to different screen sizes and devices, ensuring optimal user experience on desktops, tablets, and mobile phones. They use fluid grids, flexible images, and media queries to adjust layouts based on the user’s device.

Customisable Layouts

Customizable layouts allow users or developers to modify the design and structure of a website easily, tailoring it to specific preferences or needs. This flexibility is commonly achieved through CSS frameworks or templates that offer customization options without extensive coding.
